# LyriCalibertions Supplies Ecosystem

A comprehensive Flutter application for Lyri Calibrations Supplies Ecosystem - a complete e-commerce and service booking platform.

## Important Notice
**This project should not be copied.** This is a proprietary project owned by mrbuddhu and is not intended for distribution or reproduction.

## Features

- **User Authentication**: Login, Signup, OTP verification, Password reset
- **Product Management**: Browse products, View product details, Product listings
- **Service Booking**: Book calibration services, View service details
- **Order Management**: View orders, Track order status
- **Profile Management**: Edit profile, Manage user information
- **Payment Integration**: Stripe payment processing
- **Safety Equipment**: Browse and purchase safety equipment
- **Reports**: Access reports and analytics
- **Notifications**: Stay updated with app notifications
- **Help & Support**: Get assistance and support

## Technologies Used

- **Flutter**: Cross-platform mobile development
- **Firebase**: Backend services including Authentication, Realtime Database, and Storage
- **Stripe**: Payment processing integration
- **GetX**: State management and navigation
- **Google Fonts**: Custom typography
- **Image Picker**: Image selection and upload

## Project Structure

```
lib/
├── helpers/          # Constants, utilities, and helper classes
├── models/           # Data models
├── services/         # Firebase and other services
└── views/            # UI screens
    └── screens/      # All application screens
```
